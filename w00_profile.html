사용자 등록관리

users 테이블 기준 사용자 등록, 조회, 수정, 상태 및 권한 관리를 수행합니다.

신규
사용자 정보 * 필수 입력
Grid 기반 사용자 등록관리 화면입니다. 신규 사용자는 팝업에서 func=signup_user로 등록하고, 등록 후 사용자ID로 자동 조회합니다.
기존 ProfileLite 세로형 입력 구조를 HiveGridLite 기반 다열 Grid 구조로 변경했습니다. 신규 등록은 signup.php의 입력 흐름을 팝업으로 반영하여 사용자ID, 비밀번호, 비밀번호 확인, 이름, 이메일을 입력하고 등록 후 즉시 조회합니다.

신규 사용자 등록

신규 등록은 기본 가입 흐름과 동일하게 사용자ID, 비밀번호, 비밀번호 확인, 이름, 이메일을 입력합니다. 등록이 완료되면 사용자ID로 관리 화면에 자동 조회됩니다.
